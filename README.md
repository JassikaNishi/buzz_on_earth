# BuzzOnEarth – Smart Bus Route & Maintenance System

This project was built for the **BuzzOnEarth India Hackathon 2024**, hosted at **IIT Kanpur**, and selected among the top 20 finalists out of 2,200+ teams.

## 🔍 Overview

A sustainable mobility solution to:
- Optimize public bus routes
- Forecast weekly crowd patterns
- Predict vehicle maintenance needs

## 🧠 Tech Stack
- **Languages**: Python
- **Libraries**: Pandas, Scikit-learn, Matplotlib
- **Web**: Flask (for local interface)

## 📁 Modules
- `bus-route/`: Route optimization
- `travell-pattern/`: Crowd prediction using ML
- `maintenance/`: Predictive maintenance
- `interface/`: Flask app for interaction
